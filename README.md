

## Introduction
This is a code repository for the corresponding video tutorial. 

In this video, we will create a Google Clone app. We're going to use React and Goole Search API powered by https://rapidapi.com.



Setup:
- Create a free account on RapidAPI.com and subscribe to Google Search API, copy Rapid API tokens into StateContextProvider , test and deploy with netlify 
- run ```npm i && npm start``` to start the development server
